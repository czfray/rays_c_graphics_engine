01 Initialisation and Window Creation

This section describe how to start using RCGE.

1.1 Initialisation

To use RCGE, you need to initialise it.

Call rcge_init at the very start of the program. That's it.

Window Creation

After initialisation, creating an application window is the next most important thing. A window is much needed as it is where graphics are actually rendered to.

To create a window, rcge_window_create needs to be called.

An example for window creation:

rcge_window window = rcge_window_create(800, 600, "RCGE Test", true); //800x600 resizable window named "RCGE Test".

Run the window

After window creation, you need to actually run the window to use it. To do that, you need to run rcge_window_run.

An example for window running:

rcge_window_run(window, start, update, resize); //run the window
rcge_window_run
void rcge_window_run(rcge_window window, rcge_start_callback start_cb, rcge_update_callback update_cb, rcge_resize_callback resize_cb)
Starts to run an application window.

The 2nd, 3rd and 4th parameters are user-defined callbacks. In this example, I have defined the following:

void start(rcge_window window)
{
//Called when function first runs
}
void update(rcge_window window, float delta_time)
{
//Called every frame update
}
void resize(rcge_window window, int width, int height)
{
//Called when window resizes
}

The 2nd parameter is type rcge_start_callback. You can define and input a function with parameter (rcge_window window) and no return. The function would be automatically called when the window is first operational. In my example, function start is called when the window starts.

The 3rd parameter is type rcge_update_callback. You can define and input a function with parameter (rcge_window window, float delta_time) and no return. The function would be automatically called for every frame update. In my example, function update is called every frame.

The 4th parameter is type rcge_resize_callback. You can define and input a function with parameter (rcge_window window, int width, int height) and no return. The funtion would be automatically called everytime the window is resized. In my example, function resize is called everytime the window resizes.
