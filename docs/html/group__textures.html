Textures

Detailed Description

Image texture management.

This module allows the creation and usage of image textures. Image textures can be rendered on meshes.

rcge_texture rcge_texture_create ( char * path,
rcge_texture_wrap_type wrap,
rcge_texture_filter_type filter )

Creates an image texture from an image file.

Parameters
[in]pathRelative path of the image file to be used for creating the texture.
[in]wrapThe wrapping method to use for the texture. See rcge_texture_wrap_type for what can be put here.
[in]filterThe filter method to use for the texture. See rcge_texture_filter_type for what can be put here.
Returns
The texture created, NULL if failed.
Note
Image file is read from the bottom RIGHT pixel to the upper LEFT pixel.
