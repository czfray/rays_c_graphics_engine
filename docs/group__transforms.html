Transforms

Detailed Description

This module allows the transformations of graphical 2D/3D objects. A transform is a transformation management object that allows the translation, rotation and scaling of objects.

Note
In RCGE, it is defined that +X is LEFTWARDS, +Y is UPWARDS, and +Z is FORWARD.
