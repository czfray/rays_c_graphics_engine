Creates a single mesh element (an element with a single mesh component inside it), then attach the newly created element to a specified manager.

Parameters
[in]managerElement manager the newly created element to be attached to.
[in]shaderShader the mesh component uses.
[in]textureThe texture of the mesh component.
[in]verticesThe vertex data array for the mesh component.
[in]vertices_sizeThe array size (number of floats) of vertices.
[in]indicesThe index data array for the mesh component.
[in]indices_sizeThe array size (number of integers) of indices.
[in]colorThe color you want to multiply to the mesh component.
[in]draw_typeDraw frequency type of the mesh component. See rcge_mesh_draw_type to see which one to choose.
[out]indexThe index of the element created in the manager. NULL if not needed.
Returns
The element created, NULL if failed.
Note
Ensure the intended shader is currently active before creating the mesh by using rcge_shader_use.
