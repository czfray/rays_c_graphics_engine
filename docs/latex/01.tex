This section describe how to start using RCGE.\hypertarget{01_init}{}\doxysection{\texorpdfstring{Initialisation}{Initialisation}}\label{01_init}
To use RCGE, you need to initialise it.

Call rcge\+\_\+init at the very start of the program. That is it.\hypertarget{01_window}{}\doxysection{\texorpdfstring{Window Creation}{Window Creation}}\label{01_window}
After initialisation, creating an application window is the next most important thing. A window is much needed as it is where graphics are actually rendered to.

To create a window, \doxylink{group__windows_ga2a9a3a1f4b8108c5f964c28c666087da}{rcge\+\_\+window\+\_\+create} needs to be called.
